# CLIMB - COMMAND LINE MANDELBROT
Simplistic Mandelbrot explorer using ncurses

## PREREQUISITES
	ncurses

## BUILDING
	make

## MANUAL
+ *ENTER*       - zoom in
+ *SPACEBAR*    - zoom out
+ *ARROW KEYS*  - scroll around
+ *q*           - quit
+ *a*           - toggle ASCII
+ *h*           - show more information
